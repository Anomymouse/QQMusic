# QQMusic
高仿QQ音乐播放界面


# HYKQQMusicDemo

- 素材提取为安卓apk包
- 使用SSizeClass进行横竖屏布局
- 使用AVFoundation进行音频播放管理
- 使用正则匹配歌词解析类
- 重绘了UIlabel的颜色
- 注册后台播放和远程控制事件
